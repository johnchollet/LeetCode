50. Pow(x, n)

Implement pow(x, n), which calculates x raised to the power n (xn).

Example 1:

Input: 2.00000, 10
Output: 1024.00000

Example 2:

Input: 2.10000, 3
Output: 9.26100

Example 3:

Input: 2.00000, -2
Output: 0.25000
Explanation: 2-2 = 1/22 = 1/4 = 0.25

Note:

  • -100.0 < x < 100.0
  • n is a 32-bit signed integer, within the range [−231, 231 − 1]
C++ 代码:
class Solution {
public:
double myPow(double x, int n) {
ios::sync_with_stdio(false);
cin.tie(0);
if (n == 0) return 1;
if (n < 0) {
if (n == INT_MIN) return (double)(x < 0 ? -1.: 1.) / myPow(x, -(n+1));
return 1. / myPow(x, -n);
}
double half = myPow(x, n >> 1);
if (n&1){
return half * half * x;
}
else{
return half * half;
}
}
};
思路：x^n 次方，对 n 进行折半迭代：x^n = (x^(n/2))^2。如果 n 为奇数，那么还要多乘以一个 x 本身。对于负数，先变成正数算完，再求倒数即可
边界情况: 负数 INT_MIN -2^31 变成正数后超界了，所以要单独处理
复杂度: 时间 O(logn)，空间 O(logn) (考虑树的高度)
