7. Reverse Interger

Given a 32-bit signed integer, reverse digits of an integer.

Example 1:

Input: 123
Output: 321

Example 2:

Input: -123
Output: -321

Example 3:

Input: 120
Output: 21

Note:
Assume we are dealing with an environment which could only store integers within the 32-bit signed integer range: [−231,  231 − 1]. For the purpose of this problem, assume that your function returns 0 when the reversed integer overflows.

1. C++ 解法：
class Solution {
public:
int reverse(int x) {
ios::sync_with_stdio(false);
cin.tie(0);
int res = 0;
while(x!=0){
if (abs(res) > INT_MAX / 10) return 0;
res = res * 10 + x % 10;
x /= 10;
}
return res;
}
};
需要注意的几点：
(1) ios::sync_with_stdio(false); 和 cin.tie(0);
不加这两行是 16 ms, 加了是 8 ms;
原因: cin，cout之所以效率低，是因为先把要输出的东西存入缓冲区，再输出，导致效率降低，而第一个语句可以来打消 iostream 的输入输出缓存，可以节省许多时间，使效率与scanf与printf相差无几。第二句呢，在默认的情况下cin绑定的是cout，每次执行 << 操作符的时候都要调用flush，这样会增加IO负担。可以通过tie(0)（0表示NULL）来解除cin与cout的绑定，进一步加快执行效率。参考：http://www.cnblogs.com/PrayG/p/5749832.html
(2) if (abs(res) > INT_MAX / 10) return 0;
这一句要在 res 计算那行之前先判断，如果放在后面，溢界时还没到判断语句就 gg 了，这一行的解释可参考：http://www.cnblogs.com/grandyang/p/4125588.html
2. Python 解法：
class Solution(object):
def reverse(self, x):
"""
:type x: int
:rtype: int
"""
res = 0;
if x < 0:
neg = -1;
else:
neg = 1;
x = neg * x;
while (x!=0):
res = res * 10 + x % 10
if not - 2**31 <= res <= 2**31 - 1:
return 0
x = x / 10
return res * neg
